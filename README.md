# atcoder-docker

## 起動方法

```
docker compose up

docker compose exec atcoder bash

```

## ログイン

```
acc login

oj login https://beta.atcoder.jp/
```

## 問題取得

```

acc new {ID}

```

http://tatamo.81.la/blog/2018/12/07/atcoder-cli-tutorial/

## テスト

```
cd {ID}

cd {問題番号}

npm run test
```

## 問題提出

```
cd {ID}

cd {問題番号}

npm run submit
```
